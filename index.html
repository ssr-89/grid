repeat()

gap

grid-column-gap: 30px; - отступы между КОЛОНКАМИ

grid-row-gap: 20px; - отступы между РЯДАМИ

grid-gap: 50px 10px; - общее свойство по отступам. первое между рядами, второе между колонками

gap: 50px 10px; - также можно указать просто gap и прописать одно два значения.

Одно значение оно назначается и рядам и колонкам



grid-template-columns: 70px 70px 70px 70px 70px;

gap: 10px 30px;

Перемещение ячеек внутри сетки (по колонкам)

Дочернему элементу задаётся: grid-column-start: 1; grid-column-end: 5;

Или сокращённо: grid-column: 1 / 5; 

.grid-9 > div:nth-child(1) {
  grid-column-start: 1;
  grid-column-end: 5; // или -1

  // grid-column: 1 / 5; - сокращённо
}

Области сетки grid areas

<div id="wrapper" class="wrapper">
  <header>HEADER</header>
  <aside>SIDEBAR</aside>
  <main>MAIN</main>
  <footer>FOOTER</footer>
</div>

// родитель
.wrapper {
  grid-template-columns: 200px 1fr;
  grid-template-rows: 100px 1fr 100px;

  grid-template-areas:
    "header header"
    "aside main"
    "aside footer";
}

// дочерние элементы
.wrapper > header {
  grid-area: header;
}
.wrapper > aside {
  grid-area: aside;
}
.wrapper > main {
  grid-area: main;
}
.wrapper > footer {
  grid-area: footer;
}
